# oss_project

> 2021204016 윤희성

A personal introduction webpage built with FastAPI and vanilla JavaScript,  
created as part of an Open Source Software course project.

---

## 📌 About

A self-introduction website developed from scratch using open-source technologies.  
The page is served via FastAPI on the backend and styled with plain HTML and CSS on the frontend.  
Deployed using nginx on Ubuntu.

---

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Python, FastAPI |
| Server | nginx, Ubuntu |
